This G711 encode example is used to encode PCM data to G711 data, and then decode the data with G711 decoder. We can hear the sound from audio jack. 
In order to run the example the following steps must be followed.

	1. Set the parameter CONFIG_EXAMPLE_AUDIO_TO_G711 to 1 in platform_opts.h file

	2. Choose the G711 encoder (encodeU or encodeA)
	
	3. Build and flash the binary to test

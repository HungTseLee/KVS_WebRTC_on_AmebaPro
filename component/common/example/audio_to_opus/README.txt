This OPUS encode example is used to encode PCM data to OPUS data, and then decode the data with OPUS decoder. We can hear the sound from audio jack. 
In order to run the example the following steps must be followed.

	1. Set the parameter CONFIG_EXAMPLE_AUDIO_TO_OPUS to 1 in platform_opts.h file
	
	2. Build and flash the binary to test

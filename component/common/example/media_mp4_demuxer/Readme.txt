This is an example to seperate the video and audio from mp4 file.

How to use the example
1.Put the mp4 file into the sd card and run the example.The file name is named AMEBA_PRO.mp4.
2.Pull out the SD CARD after the example is finished.
3.It will save the ameba_video.h264 and ameba_audio.aac. We can play the video and audio from potplayer.

Note:

It only support mp4 files for Ameba Pro.

This audio loop example is used to pass PCM rx data to PCM tx data directly. We can hear the sound from audio jack. 
In order to run the example the following steps must be followed.

	1. Set the parameter CONFIG_EXAMPLE_AUDIO_LOOP to 1 in platform_opts.h file
	
	2. Build and flash the binary to test

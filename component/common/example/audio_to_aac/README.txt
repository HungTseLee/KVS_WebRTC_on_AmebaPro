This AAC encode example is used to encode PCM data to AAC data, and then save a 10 seconds AAC data to SD card. 
In order to run the example the following steps must be followed.

	1. Set the parameter CONFIG_EXAMPLE_AUDIO_TO_AAC to 1 in platform_opts.h file

	2. Set the AAC codec parameter
	
	3. Build and flash the binary to test

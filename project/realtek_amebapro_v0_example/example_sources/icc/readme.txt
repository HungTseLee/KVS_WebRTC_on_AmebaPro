Example Description

This example shows:
1. How to send ICC commands from HS/LS platform to the LS/HS platform.
2. A ICC message loop back. Icc message data is sent from HS to LS and then LS loop back ICC message to the HS.

The source file "hs_main.c" is the test code should be run on the HS platform. The source file "ls_main.c" is the test code should be run on the LS platform.

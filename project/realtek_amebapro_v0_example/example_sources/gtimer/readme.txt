Example Description

This example describes how to use general timer.

Requirement Components:
    2 LED

HS
    Connect the two LED to port PH_0 and PH_1 respectivly. 

LS
    Connect the two LED to port PA_2 and PA_3 respectivly. 

Behavior:
The two LED will blink at different frequence.

    Two timers are intialized in this example
    (1) Periodic timer
    (2) One shut timer
